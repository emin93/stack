install.emin.ch

One command to provision a fresh Apple Silicon Mac.

bash -c "$(curl -fsSL https://install.emin.ch/install.sh)"

Paste into Terminal on a fresh Mac. You'll be guided through every sign-in (GitHub, Claude Code, Codex, Apple ID). The same command on an existing setup is safe — it adds missing pieces and bumps versions.

What gets installed

  • Xcode Command Line Tools + latest Xcode (via xcodes)
  • Homebrew, then everything in the Brewfile: ripgrep, fd, gh, ffmpeg, mise, stow, aria2, JetBrains Mono
  • Casks: Ghostty, 1Password, Brave, Claude Code, Codex
  • mise with global Node LTS and latest pnpm
  • Dotfiles symlinked via stow: zsh, git, Ghostty
  • A new ed25519 SSH key, uploaded to your GitHub account

What you still do by hand

  • Fill in ~/.gitconfig.local with your name and email
  • Sign in to apps with GUI-only auth (1Password, Brave)
